우리 가족 홈페이지 실행 방법

1. family_homepage.zip 파일의 압축을 풉니다.
2. 압축을 푼 폴더 안의 index.html 파일을 더블클릭합니다.
3. 가족 사진, 가족 소개, 사진첩, 채팅 내용을 입력해 사용합니다.

중요 안내
- 이 버전은 서버 없이 PC에서 바로 실행되는 정적 홈페이지입니다.
- 입력한 내용은 현재 사용하는 브라우저의 localStorage에 저장됩니다.
- 다른 가족이 각자 다른 휴대폰에서 동시에 채팅하려면 Firebase, Supabase, 카카오 로그인, 서버 호스팅 등이 추가로 필요합니다.
- 친구나 친지에게 보여주려면 폴더 전체를 압축해서 전달하거나, GitHub Pages / 카페24 / 웹호스팅에 올리면 됩니다.


수정사항
- 가족 사진과 사진첩 사진이 잘리지 않고 전체가 보이도록 수정했습니다.
- 사진 비율이 서로 다를 경우, 남는 공간은 배경색으로 표시됩니다.
